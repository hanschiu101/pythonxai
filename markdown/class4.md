## 一、class4-1：用 Streamlit 顯示圖片 🖼️

### 這一課在學什麼？

👉 **把圖片放到網頁上顯示出來**

### 重點說明

- `st.title()`：顯示大標題
- `st.image()`：顯示圖片
- `width`：圖片的寬度
- `caption`：圖片下面的說明文字

### 範例在做的事

- 顯示 🍎 蘋果、🍌 香蕉、🍊 橘子的圖片
- 從「image」資料夾中：
  - 找出所有圖片
  - 一張一張顯示出來

### 小學生記法 🧠

> 📂 資料夾裡有很多圖片
> 🔁 用迴圈把圖片一張一張秀出來

## 三、class4-3：什麼是「函數」 🔧

### 函數是什麼？

👉 **可以重複使用的小工具**

就像：

- 微波爐：丟進去 → 按按鈕 → 出來
- 函數：給資料 → 執行 → 得到結果

---

### 1️⃣ 最簡單的函數

```python
def hello():
    print("hello")
```

👉 每次叫它，就印一次 `hello`

---

### 2️⃣ 有參數的函數

```python
def greet(name):
    print(f"Hello {name}!")
```

👉 名字不一樣，結果也不一樣

---

### 3️⃣ 回傳結果的函數

```python
def two_num_min(a, b):
    return 比較小的數
```

👉 幫你找出比較小的數字

---

### 4️⃣ 有「預設值」的函數

```python
def calarea_circle_area(radius, pi=3.14):
```

👉 如果沒有給 `pi`，就用 3.14

---

### 小學生記法 🧠

> 🧰 函數 = 小工具
> 📥 放資料進去
> 📤 拿結果出來

## 四、class4-4：全域變數 & 區域變數 🌍🏠

### 什麼是變數？

👉 用來裝數字或資料的盒子

---

### 🌍 全域變數

- 在「外面」定義
- 大家都看得到

```python
length = 5
```

---

### 🏠 區域變數

- 在函數「裡面」定義
- 只有函數裡面能用

```python
def calculate():
    area = length * length
```

---

### ⚠️ 常見重點

- 函數裡的變數，外面通常不能用
- 如果要在函數裡改外面的變數：
  - 要用 `global`

```python
global area
```

---

### 小學生記法 🧠

> 🌍 全域：大家都能用
> 🏠 區域：只有家裡能用
> 🚫 不小心會出錯，要注意
