## 🧺 一、購物籃系統（Streamlit）

### 這是在做什麼？

用 **Streamlit** 做一個「可以加商品、刪商品的購物籃」。

### 重點概念

- `st.title()`：顯示標題
- `st.session_state`：
  👉 像「記憶盒子」，可以記住購物籃裡的東西
- `st.text_input()`：輸入商品名稱
- `st.button()`：按鈕
- `append()`：把商品放進購物籃
- `pop()`：把商品刪掉

### 小學生版本理解

- 購物籃 = 一個裝東西的箱子 🧺
- 輸入商品 → 按「加入商品」
- 商品會出現在清單裡
- 按「刪除」 → 商品不見了

## ➕ 二、算術指定運算（快速算數）

這些是「**一邊算，一邊存起來**」的寫法。

| 寫法      | 意思               |
| --------- | ------------------ |
| `a += b`  | a = a + b          |
| `a -= b`  | a = a - b          |
| `a *= b`  | a = a × b          |
| `a /= b`  | a = a ÷ b          |
| `a //= b` | a = 只留整數的除法 |
| `a %= b`  | a = 餘數           |
| `a **= b` | a = 次方           |

### 小例子

```python
a = 10
a += 3
```

👉 a 變成 **13**

## 🔁 三、while 迴圈（一直做，直到不行）

### 基本 while

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

👉 從 0 印到 4

### break（強制停止）

```python
if i == 3:
    break
```

👉 碰到 3 就停 ✋

### 小學生理解

- while 就像：「**只要條件對，就一直做**」
- break 就像：「**我不要玩了，停！**」

---

## 🔂 四、for 迴圈（照次數做）

```python
for i in range(1, 6):
    print(i)
```

👉 印出 1～5

- `range(1,6)` = 1,2,3,4,5
- 也可以搭配 `break` 停下來

## 🎲 五、random 隨機數

### randrange

```python
random.randrange(1, 6)
```

👉 會出現 **1～5 的其中一個**

```python
random.randrange(0, 10, 2)
```

👉 只會出現 **0,2,4,6,8**

### randint

```python
random.randint(1, 5)
```

👉 1～5（包含 1 跟 5）

### 統計出現次數

- 用 **for 迴圈跑很多次**
- 用 **list 記錄次數**

👉 看電腦「抽籤」公平不公平 🎲

## 🎯 六、猜數字遊戲（Streamlit）

### 遊戲規則

- 電腦先想一個 0～100 的數字
- 你來猜
- 電腦會說：
  - 太小了
  - 太大了
  - 猜對了 🎉

### 用到的重點

- `session_state`：記住答案
- `number_input()`：輸入數字
- `if / elif / else`：判斷對錯
- `st.balloons()`：猜對放氣球 🎈

## 📦 七、字典 dict（像資料表）

### 什麼是 dict？

👉「**一個名字，對一個資料**」

```python
scores = {"國語": 90, "英文": 80}
```

### 取資料

```python
scores["國語"]  # 90
```

### 走訪字典

- `keys()`：看名字
- `values()`：看數字
- `items()`：名字＋數字一起看
