# 🧠 今天學了什麼？（小學生版）

---

## ① Streamlit 是什麼？ 🖥️✨

👉 **可以把程式變成有畫面、可以點的網頁**

🧒 想像成：
寫程式 ≈ 在做一個小遊戲或小網站
可以：

- 輸入數字
- 按按鈕
- 看到氣球、下雪 ❄️🎈

---

## ② 輸入數字（number_input） 🔢

```python
st.number_input("請輸入一個數字")
```

📌 功能：

- 讓使用者在畫面上輸入數字
- 可以限制最小、最大值

🧒 像是：

> 螢幕上出現一個「數字輸入盒」

---

## ③ 成績等第判斷（if 判斷） 📊

```python
if 分數 >= 90:
    A
elif 分數 >= 80:
    B
```

📌 功能：

- 電腦會幫你判斷成績是 A、B、C…

🧒 小朋友版：

- 如果考很好 → A
- 沒那麼好 → B、C、D
- 不及格 → F

電腦會**一個一個檢查**

---

## ④ 重複做事情（for 迴圈） 🔁

```python
for i in range(5):
    print("hello")
```

📌 功能：

- 讓電腦重複做一樣的事

🧒 像是：

- 說 5 次 hello
- 數數字：0、1、2、3、4
- 每次跳 2：2、4、6、8

---

## ⑤ 按鈕＋特效 🎈❄️

```python
if st.button("點我"):
    st.balloons()
```

📌 功能：

- 按下按鈕，畫面就會有反應

🧒 像是：

> 按按鈕 → 變魔術 🎩
> 氣球飛出來、開始下雪！

## ⑥ 用星星畫圖（箭頭 ⭐）

📌 功能：

- 用 `*` 和空格畫出箭頭形狀
- 可以自己決定高度

🧒 想像成：

> 用文字在畫畫，像在拼積木 🧱

## ⑦ 串列（List）📦

### 1️⃣ 什麼是串列？

```python
[1, 2, 3]
["apple", "banana"]
```

📌 功能：

- 一個箱子，可以裝很多東西

🧒 可以裝：

- 數字
- 文字
- 不同種類
- 箱子裡還有箱子！

---

### 2️⃣ 拿出串列裡的東西

```python
l[0]
```

📌 規則：

- 第一個位置是 **0**
- 不是 1 ❗

🧒 像是：

> 第一個抽屜編號是 0

---

### 3️⃣ 切片（拿一部分）

```python
l[1:4]
```

🧒 像是：

> 只拿中間幾個，不是全部

---

### 4️⃣ len() 看有幾個

```python
len(l)
```

📌 功能：

- 看箱子裡有幾樣東西

---

### 5️⃣ 一個一個看（for 走訪）

```python
for element in l:
    print(element)
```

🧒 像是：

> 老師一個一個點名 👩‍🏫

---

### 6️⃣ 串列可以被改 ✏️

```python
a[0] = 10
```

⚠️ 注意：

- 如果兩個名字指到同一個箱子
  👉 改一個，另一個也會變！

---

### 7️⃣ 串列常用功能 🛠️

- `append()`：加東西
- `remove()`：移除指定東西
- `pop()`：拿走一個
- `sort()`：排順序
